# Step 3: Review Summary

## Task

Summarize the code review findings.

### Create Summary

1. **Overall Assessment**
   - Approve / Request Changes / Comment
   - Overall code quality rating (1-5)

2. **Critical Issues** (must fix before merge)
   - List any blocking issues

3. **Recommendations** (should fix)
   - List important but non-blocking issues

4. **Suggestions** (nice to have)
   - List minor improvements

5. **Positive Feedback**
   - Highlight well-written code
   - Good patterns used

## Output Format

```
## Review Summary

**Overall: [Approve/Request Changes/Comment]**
**Quality: [X/5]**

### Critical Issues
- [ ] Issue 1
- [ ] Issue 2

### Recommendations
- [ ] Recommendation 1

### Suggestions
- Suggestion 1

### Positive Notes
- Good job on X
```
